코드업 [1063번] (https://codeup.kr/problem.php?id=1063) 문제를 해결하는 중 파이썬에서는 삼항 연산자의 형태가 다른 언어들과 다르다는것을 알게되었다.


#기존 언어들
```
조건식 ? 참 일때의 값 : 거짓 일때의 값
```


#파이썬
```
참 일때의 값 if 조건식 else 거짓 일때의 값
```
